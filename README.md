# finance-dashboard

個人向け財務管理Webアプリです。収入、支出、Uber収入、資産推移、確定申告用経費を1つのダッシュボードで管理する初期版です。

## 主な機能

- 収入管理: 給与・副業・投資収入を一覧化
- 支出管理: 生活費や固定費をカテゴリ別に管理
- Uber収入管理: 週別売上、チップ、手数料、純収入を可視化
- 資産推移グラフ: 現金・投資を含む総資産の推移を表示
- 確定申告用経費管理: 経費対象支出を抽出し年間見込みを計算

## 開発

```bash
npm install
npm run start
```

## ビルド

```bash
npm run build
```
