# finance-dashboard

個人向け財務管理Webアプリです。収入、支出、Uber収入、資産推移、確定申告用経費を1つのダッシュボードで管理できます。

## 主な機能

- 収入追加: 日付・内容・カテゴリ・金額を入力して保存
- 支出追加: 日付・内容・カテゴリ・金額・確定申告用経費フラグを入力して保存
- Uber収入追加: 週別の配達件数、売上、チップ、手数料を入力して保存
- 資産推移自動計算: 初期資産に月別の収入、Uber純収入、支出を反映してグラフ化
- LocalStorage保存: ブラウザを再読み込みしても入力済みデータを保持

## 開発

```bash
npm run start
```

## ビルド・検証

```bash
npm run build
```
